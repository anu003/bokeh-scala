# bokeh-scala

Scala bindings for Bokeh plotting library.

## Requirements

* [Java](http://wwww.java.com) JRE 1.6+

## Usage

```bash
$ git clone https://github.com/mattpap/bokeh-scala
$ cd bokeh-scala
$ ./sbt
> examples/runMain org.continuum.bokeh.examples.Anscombe
```
